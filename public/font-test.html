How to Use These Fonts

You can now use these fonts in your CSS with the following CSS custom properties:

  • var(--font-family-din) - DIN-2014 Regular family
  • var(--font-family-din-condensed) - DIN-2014 Condensed family
  • var(--font-family-din-narrow) - DIN-2014 Narrow family

Or directly in CSS:

  • font-family: "DIN-2014", sans-serif;
  • font-family: "DIN-2014-Condensed", sans-serif;
  • font-family: "DIN-2014-Narrow", sans-serif;
